# Django_ttsx

主要说明：使用Django框架实现天天生鲜网站的商品展示、商品分类展示、商品详细、购物车、订单提交、订单成功等显示页面的前后端交互，另外实现了分页显示、admin管理后台表和用户注册与登录机制等功能。

## 项目环境

- Linux（Centos7) 

- Mysql (v5.7.13) 

- Django (v2.2.13)

## 项目页面

该天天生鲜项目分为7个页面, 分别是：

1. 首页商品展示页面
2. 商品分类展示页面
3. 商品详细页面
4. 购物车页面
5. 订单提交界面
6. 订单成功显示页面
7. 用户注册与登陆页面
